# 🧮 Basit Hesap Makinesi

Bu projede kullanıcı, HTML ve JavaScript ile oluşturulmuş bir hesap makinesi arayüzü üzerinden toplama, çıkarma, çarpma ve bölme işlemlerini gerçekleştirebilir.

## 📌 Özellikler

- Rakam butonları (0-9) ile sayı girişi
- Dört işlem butonları: +, -, *, /
- = tuşu ile sonucu hesaplama
- C (Clear) tuşu ile ekranı sıfırlama
- Basit ve kullanıcı dostu arayüz
